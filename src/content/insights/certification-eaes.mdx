# Сертификация мебели и оборудования по ТР ЕАЭС: что нужно знать

Импорт мебели в страны ЕАЭС требует обязательной сертификации. Разбираем, какие документы нужны и как избежать задержек на таможне.

## Технические регламенты ЕАЭС для мебели

**ТР ЕАЭС 025/2012** — основной регламент для мебели. Распространяется на:

- Мебель для жилых помещений
- Мебель для общественных мест
- Детскую мебель (повышенные требования)

## Виды документов

### Декларация соответствия
Для большинства видов мебели достаточно декларации, которую принимает производитель или импортёр.

### Сертификат соответствия
Обязателен для детской мебели и мебели с упholstery из полимерных материалов.

## Документы для таможни

1. Декларация соответствия или сертификат (с номером и датой)
2. Протоколы испытаний аккредитованной лаборатории
3. Эксплуатационная документация на русском языке
4. Инструкция по сборке и применению

## Сроки оформления

| Документ | Срок |
|----------|------|
| Декларация соответствия | 5–10 рабочих дней |
| Сертификат соответствия | 15–30 рабочих дней |
| Пожарный сертификат | 20–40 рабочих дней |
| СЭЗ (медицина) | 30–60 рабочих дней |
