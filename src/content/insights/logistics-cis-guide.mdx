# Логистика и таможня СНГ: полный гид для проектных поставок

Поставка мебели из Европы в Ташкент — это не просто отправить контейнер. Это таможенное оформление в трёх странах, документация на 12 языках и понимание ТН ВЭД кодов.

## Маршруты поставок

### Из Европы в Узбекистан

**Морской путь**: Европа → Стамбул/Мерсин → Туркменбаши → Ташкент. Срок: 35–45 дней.

**Авиа**: Для срочных и высокостоимостных позиций. Срок: 3–5 дней. Стоимость ×4–6.

**Авто**: Через Россию или Иран. Срок: 12–18 дней. Ограничения по некоторым маршрутам.

## Таможенное оформление в Узбекистане

Для мебели и оборудования применяются ТН ВЭД коды группы 94. Ключевые ставки:

| Категория | Ставка |
|-----------|--------|
| Мебель мягкая | 15% |
| Мебель офисная | 10% |
| Освещение | 10% |
| Кухонное оборудование | 5% |

## Документы для импорта

- Инвойс (коммерческий и проформа)
- Упаковочный лист
- Сертификат происхождения
- CMR / коносамент
- Сертификаты соответствия
