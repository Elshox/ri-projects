## 🚀 Шаги работы

### Шаг 1. Подготовка окружения (один раз)

На своей машине нужны:
- Node.js 20+
- npm или pnpm
- Claude Code (установлен локально)
- Аккаунты Vercel + GitHub
- Аккаунт Resend (для email)
- Telegram-бот (создаётся за 2 минуты у @BotFather)

### Шаг 2. Создание проекта

Открой терминал:

```bash
# Создай папку и зайди в неё
mkdir ~/projects/ri-projects
cd ~/projects/ri-projects

# Запусти Claude Code в этой папке
claude
```

### Шаг 3. Первая команда Claude Code

Положи в папку проекта файл `CLAUDE.md` (из этого пакета).

Затем дай Claude Code такой первый промпт (Промпт 0 из ТЗ):

```
Ты — senior full-stack web-разработчик, эксперт в Next.js,
TypeScript, Tailwind CSS и Motion. Я даю тебе ТЗ на разработку
премиум B2B-сайта компании RI PROJECTS.

Шаг 1: Прочитай файл CLAUDE.md в корне проекта целиком.
Это твой главный референс по бренду, токенам, структуре.

Шаг 2: Установи скилл ui-ux-pro-max:
/plugin marketplace add nextlevelbuilder/ui-ux-pro-max-skill
/plugin install ui-ux-pro-max@ui-ux-pro-max-skill

Шаг 3: Инициализируй Next.js 14 проект с TypeScript, Tailwind,
App Router, src-dir.

Шаг 4: Установи зависимости (motion, lenis, lucide-react,
shadcn/ui, react-hook-form, zod, next-intl, resend).

Шаг 5: Настрой tailwind.config.ts и globals.css с бренд-токенами
из CLAUDE.md.

Шаг 6: Создай lib/motion-presets.ts с пресетами анимаций.

Шаг 7: Настрой next-intl для RU + EN.

После каждого шага — git commit с осмысленным сообщением.
```

### Шаг 4. Дальше — по промптам из ТЗ

В документе `RI_PROJECTS_TZ_for_Claude_Code.docx`, раздел 4, есть **11 готовых промптов**. Используй их по очереди:

- Промпт 2 — Главная: Hero

- Промпт 6 — Главная: Витринные кейсы
- Промпт 7 — Главная: Партнёры (marquee)
- Промпт 8 — Главная: Процесс
- Промпт 9 — Главная: CTA + форма
- Промпт 10 — Внутренние страницы

## ✅ Контрольные точки

После каждого промпта — обязательно:

1. **Запусти `npm run dev`** и визуально проверь.
2. **Сделай git commit** с осмысленным сообщением.
3. **Прогони `npm run lint`** — нет ли ошибок.
4. **Прогони `npm run build`** — собирается ли проект.

Если что-то не так — сразу скажи Claude Code, не накапливай правки.

---

## 🚨 Если возникли проблемы

| Проблема | Решение |
|----------|---------|
| Claude не находит ui-ux-pro-max | Установи через CLI: `npm install -g uipro-cli && uipro init --ai claude` |
| Motion не работает в Next.js | Добавь `"use client"` в начало файла |
| Анимации тормозят на mobile | Скажи Claude упростить: только opacity и transform |
| Видео не грузится | Добавь poster-image как fallback |
| Форма не отправляет | Проверь env-переменные в Vercel |

## 📞 Что делать после запуска

1. Заведи аккаунты в Google Search Console + Yandex Webmaster.
2. Залей sitemap.xml.
3. Настрой Google Analytics + Yandex Metrika.
4. Сделай тестовую отправку формы.
5. Проверь все ссылки (нет 404).
6. Запусти Lighthouse — все метрики должны быть 90+.
7. Сделай OG-превью для шаринга в Telegram и LinkedIn.
